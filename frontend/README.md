# Trek Planner Frontend

React + Vite frontend for Trek Planner application.

## Setup

1. Install dependencies:
\`\`\`bash
npm install
\`\`\`

2. Start the development server:
\`\`\`bash
npm run dev
\`\`\`

The app will run on `http://localhost:3000`

## Build for Production

\`\`\`bash
npm run build
\`\`\`

## Features

- User authentication (signup/login)
- Browse treks with pagination
- Add new treks (authenticated users)
- Edit existing treks (authenticated users)
- Delete treks (authenticated users)
- Responsive design

## Tech Stack

- React 18
- React Router DOM
- Axios for API calls
- Context API for state management
- Vite for bundling
